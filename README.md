# Arcane Bot 🤖

一个功能强大的多功能 Discord 机器人，提供等级系统、欢迎/告别卡片、反应角色、计数器、日志系统等丰富功能。

[![Python](https://img.shields.io/badge/python-3.11+-green.svg)](https://www.python.org/)
[![Discord.py](https://img.shields.io/badge/discord.py-2.3.0-blue.svg)](https://github.com/Rapptz/discord.py)

## ✨ 功能特性

### 📊 等级系统
- 基于消息和语音时长的经验值系统
- 精美的等级卡片图片生成
- 等级排行榜（支持文字/语音双模式）
- 等级奖励角色自动分配
- 可调节的经验倍率

### 🎭 反应角色
- 点击 emoji 自动获得/移除角色
- 支持多条消息、多个 emoji 独立配置

### 🔢 计数器
- 实时统计服务器成员数量、在线人数
- 自动更新并发送到指定频道
- 支持自定义消息模板

### 📋 日志系统
- 消息删除/编辑记录
- 语音频道进出记录
- 管理员操作记录
- 支持分别设置不同日志频道

### 👋 欢迎/告别
- 精美的欢迎卡片（青蓝色主题）
- 离别卡片（红色主题）
- 自动显示成员序号

### 🔧 管理命令
- `/kick` - 踢出用户
- `/ban` - 封禁用户
- `/clear` - 批量清除消息
- 完整的权限控制

### ℹ️ 信息命令
- `/userinfo` - 查看用户信息
- `/help` - 查看帮助
